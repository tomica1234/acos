# ACOS Frontend

React/Vite frontend for operating ACOS.

It talks to the ACOS API through the Vite `/api` proxy during development.

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 5174
```

On Windows, this frontend is launched by:

```powershell
C:\Users\jalan\wip\acos\scripts\windows\Install-AcosScheduledTasks.ps1
```
